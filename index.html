Shiny Crime Map

Downtown Houston Texas Violent Crimes Interactive Map

James C. Fielding
JHU Developing Data Products - Project 1

Shiny Crime Map an interactive map that allows the user to view the locations of different types of violent crimes in the Houston, Texas downtown area from January 2010 through August 2010.

Shiny Crime Map is simple to use. Just select the Crime Type you would like to map in the drop down menu. You can also toggle the radio buttons to change Map Type.
